# Python Flask CI Project

This is a beginner-friendly project designed to introduce you to the fundamentals of web application development, testing, and continuous integration. You'll build a simple Flask "Hello, World!" application, write unit tests for it using Pytest, and automate the testing process with GitHub Actions.

## What You Will Learn

By working through this project, you will gain practical experience and understanding in:

*   **Flask Web Framework:** Setting up a basic Flask application and defining routes.
*   **Pytest for Unit Testing:** Writing effective unit tests to ensure your application code works as expected.
*   **Continuous Integration (CI) Principles:** Understanding the importance of automating builds and tests.
*   **GitHub Actions:** Configuring and implementing a CI workflow to automatically run tests on every code push.
*   **Development Workflow:** Best practices for setting up a local development environment with virtual environments.

## Prerequisites

Before you begin, ensure you have the following installed on your system:

*   **Python 3.x:** This project is built with Python.
    *   Verify installation: `python3 --version` or `python --version`
    *   If not installed, download from [python.org](https://www.python.org/downloads/).
*   **pip:** Python's package installer (usually comes with Python).
    *   Verify installation: `pip3 --version` or `pip --version`
*   **Git:** A version control system to clone the repository.
    *   Verify installation: `git --version`
    *   If not installed, download from [git-scm.com](https://git-scm.com/downloads).
*   **GitHub Account:** Required to interact with the repository and view GitHub Actions.

## Project Structure

The project is organized to clearly separate application code, tests, and CI/CD configurations.
